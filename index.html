      📖 User Guide

      Learn how to use ScamShield AI and stay safe online

      🚀 Quick Start (4 Steps)

      1. Copy the suspicious message – SMS, email, or WhatsApp text
      2. Paste it into the analyzer – on the Home page
      3. Click Analyze Message – wait for AI analysis (1-2 seconds)
      4. Take action based on result – report to cyber cell if scam
